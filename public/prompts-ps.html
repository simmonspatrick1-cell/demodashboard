How to Use This Guide

This guide provides ready-to-use prompts for creating NetSuite demo data through Claude. Simply copy and customize these prompts to quickly set up realistic demo scenarios for your customer presentations.

Pro Tips for Effective Demo Prep

Be specific about dates, amounts, and relationships between records Ask for records in "different stages" to show workflow progression Request "realistic" data when you want variety and authenticity Specify if you need records for specific reporting scenarios

Create interconnected data (customers → projects → tasks → invoices) to show end-to-end flows

Include records with different statuses to demonstrate approval workflows Build historical data for trend analysis and forecasting demos
